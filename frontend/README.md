# Combat Corner BD - Frontend

This is the React frontend for the Combat Corner BD blog portal.

## 🚀 Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```

## 🛠️ Tech Stack
- **React + Vite**
- **Tailwind CSS**
- **Redux Toolkit**
- **i18next** (Bilingual support)

## 📁 Key Directories
- `src/Component`: Global UI elements.
- `src/pages`: Page components and views.
- `src/redux`: State management and API hooks.
- `src/i18n.js`: UI translations.
